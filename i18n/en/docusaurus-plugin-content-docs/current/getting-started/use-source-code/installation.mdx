## Install manually {#install-manually}

1. Install [Python](https://www.python.org/) on your machine.

   :::info

   Python 版本：3.12+。

   :::

2. Download source:

   ```mdx-code-block
   <Tabs>
   <TabItem value="c1" label="下载 Release" default>
   ```

   1. Download source from [Releases](https://github.com/TTB-Network/python-openbmclapi/releases).

   2. Put downloaded source code into a pre-prepared folder.

   3. Open terminal in folder.

   ```mdx-code-block
   </TabItem>
   <TabItem value="c2" label="使用 Git 克隆">
   ```

   1. Clone repository:

      ```sh
      git clone https://github.com/TTB-Network/python-openbmclapi.git
      ```

      :::tip

      You can also speed up with mirror sources:

      ```sh
      git clone https://ghproxy.bugungu.top/https://github.com/TTB-Network/python-openbmclapi.git
      ```

      :::

   2. Open repository folder:

      ```sh
      cd python-openbmclapi
      ```

3. Install dependencies:

   :::tip

   你亦可设置 pip 镜像源来加快你在中国大陆地区的下载速度：

   ```sh
   pip config set global.index-url https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple
   ```

   :::

   ```mdx-code-block
   <Tabs>
   <TabItem value="c1" label="使用 Poetry（推荐）" default>
   ```

   1. 安装 [pipx](https://pipx.pypa.io/latest/):

      ```sh
      pip install --user pipx
      ```

   2. 重新启动终端。

   3. 安装 [Poetry](https://python-poetry.org/):

      ```sh
      pipx install poetry
      pipx ensurepath
      ```

      :::info

      pipx 需要安装 `python3.10-venv` 包来进行依赖的安装：

      ```sh
      apt-get install python3.10-venv
      ```

      :::

   4. 重新启动终端。

   5. 安装依赖：

      ```sh
      poetry install
      ```
