所有插件都储存在 `./plugins/` 文件夹。如果你想编写新的插件，请在此新建一个文件夹。

文件夹的名字并不用于区分插件名，仅用于分类。

插件会以 [Python 包](https://docs.python.org/zh-cn/3/tutorial/modules.html#packages)的形式存在，在 Python 中，需要有 `__init__.py` 文件才能让 Python 将包含该文件的目录当作包来处理，机器人在加载的时候会遍历 `./plugins/` 文件夹，并通过加载每个插件文件夹里的 `__init__.py` 来导入 Python 包。

请将想要加入的插件定义编写在 `__init__.py` 中或确保其能够被引用。
