我们假设你已经为插件编写好了代码，现在你只需要将你的代码做成能被程序识别的插件。

## 定义插件 {#define-a-plugin}

```py
__VERSION__ = "0.0.1"
__AUTHOR__ = "tianxiu2b2t"
__NAME__ = "Example Plugin"
```

通过往 `__init__.py` 增加变量即可创建一个简单的插件。

**变量说明：**

| 参数          | 类型     | 说明         |
| ------------- | -------- | ------------ |
| `__VERSION__` | `string` | 插件的版本。 |
| `__AUTHOR__`  | `string` | 插件的作者。 |
| `__NAME__`    | `string` | 插件的名称。 |
