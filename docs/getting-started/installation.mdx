# 安装

如果你想使用 Docker，请转到[使用 Docker](/docs/use-docker)。

1. 在你的机器上安装好 [Python](https://www.python.org/)。

    :::info

    建议 Python 版本：3.10+。

    :::

2. 下载源码：

    <Tabs>
    <TabItem value="c1" label="下载 Release" default>

    1. 从 [Releases](https://github.com/TTB-Network/python-openbmclapi/releases) 中下载代码。

    2. 将下载好的源码放到一个事先准备的文件夹。

    3. 在文件夹中打开终端。

    </TabItem>
    <TabItem value="c2" label="使用 Git 克隆">

    1. 克隆仓库：

        ```sh
        git clone https://github.com/tianxiu2b2t/python-openbmclapi.git
        ```

        :::info

        仓库镜像地址（长期未更新）：https://gitee.com/ttb-network/python-openbmclapi

        :::

    2. 打开仓库所在文件夹：

        ```sh
        cd python-openbmclapi
        ```
    
    </TabItem>
    </Tabs>

3. 安装依赖：

    ```sh
    pip install -r --no-deps requirements.txt
    ```

    :::tip

    你可能需要先安装 [Microsoft C++ 生成工具](https://visualstudio.microsoft.com/visual-cpp-build-tools/)。

    :::
